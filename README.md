# 🛠️ Ethical Hacking Toolkit

A collection of Python-based tools for network recon and enumeration.
Built on Kali Linux for educational purposes.

---

## 📦 Tools

| Tool | Description |
|------|-------------|
| [dns_enum](./dns_enum/) | DNS record enumerator (A, MX, TXT, SOA, AAAA) |
| [subdomain_finder](./subdomain_finder/) | Multi-threaded subdomain enumerator |

> More tools coming soon...

---

## ⚙️ Clone the Repository

```bash
git clone https://github.com/Amitkaushik1/python-hacking-toolkit.git
cd python-hacking-toolkit
```

---

## 🚀 Usage

Each tool is in its own folder with its own README. Navigate to the tool folder and follow the instructions:

```bash
cd dns_enum
python3 dns_enum.py
```

```bash
cd subdomain_finder
python3 subdomain_finder.py
```
