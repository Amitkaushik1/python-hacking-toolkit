# 🔒 PDF Protector

A simple Python tool to password-protect any PDF file using encryption.

---

## 📋 Features

- Add password protection to any PDF file
- Encrypts PDF using AES encryption
- Simple command-line interface
- Handles errors gracefully (file not found, invalid PDF, etc.)

---

## 🛠️ Requirements

- Python 3.x
- `PyPDF2` library

---

## ⚙️ Installation

**1. Clone the repository**
```bash
git clone https://github.com/Amitkaushik1/python-hacking-toolkit.git
cd python-hacking-toolkit/PDF_Protector
```

**2. Install dependencies**
```bash
pip install PyPDF2
```

---

## 🚀 Usage

```bash
python3 pdf_protector.py input.pdf output.pdf password
```

### Example

```bash
python3 pdf_protector.py document.pdf protected_document.pdf MyPassword123
```

### Output

```
Password-Protected file save as protected_document.pdf
```
